# 🎣 Fly Fishing App - Getting Your Mac Ready

Welcome to the team! Before we can start building together, you'll need to download a few apps. This guide is just the downloads—we'll walk through the rest together.

---

## Apps to Download

Install these in order. Each download link takes you straight to the installer.

---

### 1. Xcode

**What it is:** Apple's app for building iPhone apps. We need this even though we won't open it much.

**How to get it:**
1. Open the **App Store** on your Mac (the blue icon with the "A")
2. Search for **Xcode**
3. Click **Get**, then **Install**
4. Wait for it to finish (it's huge—about 12GB, so maybe start this one before lunch)
5. Once installed, open Xcode once, accept the license agreement, and let it install any additional components it asks for

---

### 2. Visual Studio Code

**What it is:** The app where you'll actually write code. Think of it like Microsoft Word, but for programming.

**Download:** [https://code.visualstudio.com](https://code.visualstudio.com)

1. Click the big blue download button
2. Open the downloaded file
3. Drag the app to your Applications folder

---

### 3. nvm (Node Version Manager)

**What it is:** Lets you switch between different versions of Node. This matters because our app needs a specific version to work correctly.

Unfortunately nvm doesn't have a click-to-install option on Mac—we'll set this up together during our first pairing session. It takes 2 minutes.

---

### 4. Node.js

**What it is:** The engine that runs our app's code behind the scenes.

**Download:** [https://nodejs.org](https://nodejs.org)

1. Click the button that says **LTS** (the "recommended for most users" version)
2. Open the downloaded installer
3. Click through the setup wizard (just keep hitting Continue/Next)

> ⚠️ **Note:** Once we set up nvm together, we'll use that to manage Node versions. Installing Node directly now just gets you started.

---

### 5. Watchman

**What it is:** A helper tool that notices when you save changes to your code.

**Download:** [https://facebook.github.io/watchman/docs/install#macos](https://facebook.github.io/watchman/docs/install#macos)

Look for the **Download the latest release** link and grab the `.pkg` file for macOS. Run the installer.

---

### 6. Git (via GitHub Desktop)

**What it is:** Tracks changes to code so we don't accidentally break things, and lets us both work on the same files.

**Download:** [https://desktop.github.com](https://desktop.github.com)

1. Download and install GitHub Desktop
2. Open it and sign in (or create a free GitHub account if you don't have one)

> This gives you Git with a visual interface instead of typing commands.
